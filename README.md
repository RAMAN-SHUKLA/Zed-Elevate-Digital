# Zed Elevate Digital - Creative Agency Website

A premium, modern React application for **Zed Elevate Digital**, a leading creative agency specializing in branding, marketing, and sophisticated digital experiences.

![Homepage Banner](./riyadidi_img.png)

## 🚀 Live Demo
Managed and hosted on Vercel: [https://zedelevatedigital.in](https://zedelevatedigital.in) (Custom Domain)
Backup URL: [https://zedelevateprojects.vercel.app](https://zedelevateprojects.vercel.app)

## ✨ Core Features
- **Dynamic Services Sections**: Interactive tiles for Social Media, Designing, Video Editing, Paid Ads, Website/SEO, and Branding.
- **Client Testimonials**: Premium slider featuring successful brand transformations.
- **Founders' Showcase**: Professional About section highlighting the partners with a clean, circular interface.
- **Vibrant Pricing Tables**: Localized pricing in ₹ with clearly defined plans (Starter, Professional, Enterprise).
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive AOS Animations**: Smooth, modern scroll-reveal effects for a premium feel.

## 🛠️ Technology Stack
- **Frontend**: React 19 (Vite)
- **Styling**: Vanilla CSS with modern Glassmorphism and Neon accents
- **Icons**: Bootstrap Icons
- **Animations**: AOS (Animate on Scroll)
- **Deployment**: Vercel

## 📁 Project Structure
- `src/pages/`: Main application routes (HomePage, AboutPage, ProjectsPage, etc.)
- `src/components/`: Reusable UI components (Navbar, ServicePage, SubServicePage)
- `src/styles/`: Centralized CSS logic (`main.css`)
- `public/`: Static assets and images

## 🚀 Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
